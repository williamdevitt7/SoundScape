SoundScape Backend API Documentation



Users:

  • View/Get (All):
    - http://18.204.82.238:5000/users
    Returns all the users in the users table as a JSON

  • Add (Signup):
    - http://18.204.82.238:5000/users/add
    Usage: ?username=__&password=__ (appended after url)
    Returns the new users data as a JSON
